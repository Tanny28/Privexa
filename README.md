## 🔥 What is Privexa?

> **Privexa** is a **privacy-first, fully offline Meeting Intelligence Engine** that transforms unstructured meeting conversations into structured, actionable organizational insights — **without any cloud dependency.**

Every word spoken in your meeting stays on your hardware. No APIs. No external calls. No surveillance.

<div align="center">

| The Old Way | ⚡ Privexa |
|:---:|:---:|
| ☁️ Cloud-dependent tools | 🔒 100% Offline AI |
| 📤 Data stored externally | 🗄️ Local encrypted storage |
| 📝 Basic summarization | 🧠 Structured intelligence + analytics |
| ❌ No conflict detection | ⚔️ Sentiment & risk engine |
| 💳 SaaS-only model | 🏢 On-prem enterprise deployment |

</div>

## ⚙️ Installation & Quickstart

### Prerequisites

```bash
# Python 3.10+
python --version

# Install Ollama for local LLM
curl -fsSL https://ollama.com/install.sh | sh

# Pull Mistral model
ollama pull mistral
```

### Clone & Setup

```bash
# Clone the repository
git clone https://github.com/Tanny28/Privexa.git
cd Privexa

# Create virtual environment
python -m venv venv
source venv/bin/activate       # Linux/macOS
# venv\Scripts\activate        # Windows

# Install dependencies
pip install -r requirements.txt
```

### Run Privexa

```bash
# Start the Ollama server (in a separate terminal)
ollama serve

# Launch Privexa core engine
python main.py

# Start FastAPI backend (optional, for dashboard)
uvicorn app:app --reload --port 8000
```

### First Meeting Analysis

```python
from privexa import MeetingEngine

engine = MeetingEngine()

# Analyze a recorded meeting
result = engine.analyze("meeting_audio.mp3")

# Access structured output
print(result.summary)
print(result.action_items)
print(result.decisions)
print(result.sentiment_report)

# Export to PDF
result.export_pdf("meeting_report.pdf")

# Export to JSON
result.export_json("meeting_data.json")
```
